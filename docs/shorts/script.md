# eluma-sst - 60 Second Script

## Hook (0-5s)
"Deine Sprachnachrichten transkribieren - komplett lokal, ohne Cloud, ohne API-Key."

## Problem (5-15s)
"Jeder kennt es: Lange Telegram-Sprachnachrichten, aber du kannst gerade nicht zuhoeren. Online-Tools? Deine Daten landen auf fremden Servern."

## Solution (15-35s)
"eluma-sst nutzt whisper.cpp direkt auf deinem Mac. Ein Befehl startet alles - Browser oeffnet sich, du siehst die App."

**Show:**
1. Terminal: `./start.sh` (2s)
2. Browser oeffnet sich mit Eluma Dark UI (3s)
3. Mikrofon-Button klicken, sprechen, Transkription erscheint (8s)
4. Telegram .ogg Datei per Drag & Drop, Ergebnis kopieren (7s)

## Call to Action (35-45s)
"Deutsch, Englisch, Franzoesisch - 5 Sprachen. Drag & Drop fuer .ogg, .mp3, .wav. Alles kostenlos."

## Close (45-60s)
"Link in der Bio. Star auf GitHub wenn es dir gefaellt. eluma-sst - Speech to Text, lokal."
